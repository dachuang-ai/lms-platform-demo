# LMS Platform Demo

## 專案名稱

智慧學習平台 Demo  
LMS Online Course Platform Demo

---

## 專案定位

這是一個由 **達創智能科技 AI** 規劃的線上課程與知識付費平台 Demo。

本 Demo 主要展示教育品牌、講師、顧問、補習班、知識型企業如何建立可上架課程、管理教師、經營會員、設計學習路徑與規劃方案權限的智慧學習平台。

此作品適合作為 LMS、線上課程平台、知識付費系統、會員學習系統的前期規劃展示。

---

## 展示重點

- 線上課程平台首頁設計
- 課程分類與精選課程展示
- 教師 / 講師管理概念
- 學習路徑規劃
- 會員登入與學員體驗
- 方案權限與訂閱方案概念
- 課程播放與學習介面概念
- 教育品牌數位化平台架構

---

## 適用客戶

此 Demo 適合以下類型客戶參考：

- 線上課程平台
- 補習班 / 教育機構
- 顧問公司
- 知識付費品牌
- 講師個人品牌
- 企業內訓平台
- 會員制學習社群
- 錄播課程與直播課程整合平台

---

## 可延伸功能

正式專案可依客戶需求延伸：

- 多教師後台
- 課程上架管理
- 會員註冊與登入
- 課程購買與付款
- 訂閱制方案
- 點數 / 優惠券 / 折扣
- 直播課程整合
- 錄播影片管理
- 學習進度追蹤
- 測驗 / 作業 / 證書
- LINE 官方帳號通知
- 金流與電子發票串接
